Welcome to the Cloud Native Vietnam community blog! This is our space to share knowledge, experiences, and insights about cloud native technologies.

## What to Expect

We'll be covering topics such as:

- **Kubernetes** — best practices, tips, and real-world use cases
- **Containers** — Docker, containerd, and container runtimes
- **Service Mesh** — Istio, Linkerd, and beyond
- **Observability** — monitoring, logging, and tracing in cloud native environments
- **CI/CD** — GitOps, Argo CD, Flux, and modern deployment pipelines
- **Platform Engineering** — building internal developer platforms

## Contributing

We welcome contributions from the community! If you'd like to write a blog post, feel free to open a pull request on our [GitHub repository](https://github.com/cloudnativevn/blogs).

### How to Write a Post

1. Create a new Markdown file in the `_articles/` directory (e.g. `my-post-title.md`)
2. Add the required front matter (layout, title, date, author)
3. Write your content in Markdown
4. Submit a pull request

We look forward to building this blog together with the community!
